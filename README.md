# Sohail Applicant-Screening ML Service (Version 2)

## Overview

This project is the second version of the Sohail Applicant-Screening Machine Learning Service developed during the AI Foundation Internship.

Version 2 improves the original prototype by introducing feature engineering, a Scikit-learn Pipeline, cross-validation, and model versioning.
